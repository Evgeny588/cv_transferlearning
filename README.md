# Cats vs Dogs с ResNet

Эксперимент по дообучению (fine-tuning) предобученной на ImageNet модели **ResNet** на датасете **cats and dogs**.  
Цель — сравнить процесс обучения и тонкости реализации в **PyTorch** и **TensorFlow/Keras**, а также закрепить понимание transfer learning в компьютерном зрении.
